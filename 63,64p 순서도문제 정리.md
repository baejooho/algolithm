## 문제 26 - 완전수 찾기 (1~200)

### 문제 설명

1부터 200까지의 **완전수**를 출력하고, 완전수의 **개수**를 최종 출력하는 알고리즘

> **완전수란?** 
> 자신을 제외한 약수의 합 = 자기 자신인 수 
> 예) 6의 약수: 1, 2, 3, 6 → 1+2+3 = 6 ✓

### 사용 변수

|변수  |역할          |
|----|------------|
|SU  |1~200까지의 자연수|
|K, J|제어 변수       |
|REM |나머지 값       |
|SUM |약수 합계       |
|CNT |완전수 개수      |

### 알고리즘 흐름

```
START
↓
REM=0, CNT=0 (초기화)
↓
SU=1 (1부터 시작)
↓
[SU > 200?] → Yes → write CNT → END
↓ No
SUM=0, K=SU/2, J=1 (약수 탐색 준비)
↓
[J > K?] → Yes → [SUM = SU?] → Yes → CNT=CNT+1, write SU
↓ No                           ↓ No
REM = SU-(SU/J)×J              ↓
↓                        SU = SU+1 (①)
[REM = 0?] → Yes → SUM = SUM+J (④)
↓ No
J = J+1 (②)
```

### 빈칸 정답

|빈칸|정답   |이유                               |
|--|-----|---------------------------------|
|① |SU+1 |SU를 1씩 증가시켜 다음 수로 이동             |
|② |J+1  |J를 1씩 증가시켜 다음 약수 탐색              |
|③ |J    |REM = SU-(SU/J)×**J** = 나머지 구하는 식|
|④ |SUM+J|J가 약수이므로 SUM에 J를 더함              |
|⑤ |SU   |약수의 합(SUM)이 SU와 같으면 완전수          |

### 핵심 포인트

- `K = SU/2` → 약수는 SU/2 이하에만 존재하므로 절반만 탐색
- `REM = SU-(SU/J)×J` → MOD 연산 (나머지 구하기)
- `REM = 0` → J가 SU의 약수임
- `SUM = SU` → 완전수 조건

## 문제 27 - 배열에서 최솟값 제외 평균 구하기

### 문제 설명

ARRAY(10) 배열에 정수 10개를 입력받고, 
**최솟값을 제외한** 나머지 9개의 평균을 구해 출력하는 알고리즘

### 사용 변수

|변수  |역할    |
|----|------|
|i, J|인덱스 변수|
|SUM |합계    |
|MIN |최솟값   |
|AVG |평균    |

### 알고리즘 흐름

```
START
↓
ARRAY(10), SUM=0, MIN=0, AVG=0 (초기화)
↓
i=1, J=1
↓
READ N → ARRAY(i)=N → [i=10?] → No → i=i+1
                      ↓ Yes
MIN=ARRAY(1), SUM=ARRAY(1) (첫번째 값으로 초기화)
↓
[(①)=10?] → Yes → SUM=SUM-MIN (④)
↓ No                ↓
J=J+1               AVG=SUM/9 (⑤)
↓                   ↓
SUM=SUM+ARRAY(J) (②) write AVG → STOP
↓
[ARRAY(J) < MIN?] → No → (루프로 돌아감)
↓ Yes
MIN=ARRAY(J) (③)
```

### 빈칸 정답

|빈칸|정답      |번호|이유                         |
|--|--------|--|---------------------------|
|① |J       |7 |J가 10이 될 때까지 반복 (배열 10개 순회)|
|② |SUM     |24|누적 합계에 ARRAY(J) 더하기        |
|③ |ARRAY(J)|18|현재 값이 더 작으면 MIN 갱신         |
|④ |SUM-MIN |26|전체 합에서 최솟값 빼기              |
|⑤ |SUM/9   |33|최솟값 제외한 9개의 평균             |

### 핵심 포인트

- 먼저 배열에 10개 입력 후 탐색 시작
- `MIN=ARRAY(1)` → 첫 번째 값을 최솟값으로 초기화
- `SUM=ARRAY(1)` → 첫 번째 값을 합계로 초기화
- J를 2부터 10까지 반복하며 MIN 갱신 + SUM 누적
- 마지막에 `SUM-MIN` 으로 최솟값 제거 후 9로 나눠 평균

## 순서도 빈칸 푸는 요령 (공통)

1. **전체 흐름 파악** → 이 알고리즘이 뭘 하는지 먼저 이해
1. **변수 역할 파악** → 각 변수가 무슨 값을 저장하는지 확인
1. **빈칸 앞뒤 흐름 확인** → “이 시점에서 뭘 해야 하지?” 생각
1. **반복문 빈칸** → 대부분 변수+1 (증가) 또는 조건 변수
1. **검증** → 선택지 넣어보고 논리적으로 맞는지 확인
